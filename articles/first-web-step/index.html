Что такое HTML?
(Hypertext markup language)

  1. Язык гипертекстовой РАЗМЕТКИ
  2. Инструмент, объясняющий браузеру, ЧТО мы хотим
  3. Сама суть сайта

Структура языка разметки HTML состоит из тегов

Что такое тег?

Условное обозначение элемента на странице. Представляет собой название тега, заключённое в угловые скобки

            
                < *название тега* >

Основные виды тегов

  1. С закрывающим тегом

    a, p, div, section, form, header, footer, h1...h6

  2. Без закрывающего тега

    img, input, link, meta

Устаревший вариант < img />

            <!DOCTYPE html>
            <html lang="en">

Аттрибуты тегов

Дополнительное значение для вашего тега

            < input type="text" value = "I am a value" >

Аттрибут id

Индивидуальный идентификатор тега. Не должен повторяться.

            < input type="text" id = "I am an id" >

Что такое CSS?
(Cascading style sheets)

  1. Язык СТИЛЕЙ
  2. Инструмент, объясняющий КАК мы хотим
  3. Улучшение внешнего вида сайта

Основу CSS составляют селекторы

Что такое селектор?

            < div class="your-class" >
            .your-class { ... } 
            < div id="your-id" >
            #your-class { ... } 
            < section >
            section { ... }

Требования к наименованию селекторов

  • Не начинается с цифры
  • На английском
  • Должен передавать логику тега, который выбирает
  • Не желательно писать длиннее трёх слов

Правила в CSS

Правило = свойство, которое мы хотим добавить к элементу (тегу), или изменить существующее

            < div class="your-class" >
            .your-class { 
    margin-top: 20px; 
    color: red; }
